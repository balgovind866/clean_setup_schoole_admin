# School Management System

A comprehensive school administration platform built with React, TypeScript, and Metronic template.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📁 Project Structure

```
src/
├── app/
│   ├── modules/          # 13 feature modules, 92 pages
│   ├── routing/          # Route configuration
│   └── pages/            # Dashboard and common pages
└── _metronic/            # Metronic template components
```

## 🎯 Modules

1. **Students** - Admission, Enrollment, Attendance, Performance, Documents
2. **Staff** - Teachers, Non-Teaching, Recruitment, Attendance, Leave, Performance
3. **Academic** - Classes, Subjects, Mapping, Calendar, Syllabus, Lessons
4. **Fees** - Structure, Collection, Tracking, Receipts, Concessions, Reports
5. **Timetable** - Class/Teacher Schedules, Resources, Substitution, Periods
6. **Examination** - Exams, Marks, Report Cards, Results, Online Exams
7. **Library** - Books, Issue/Return, Fines, Digital Resources
8. **Transport** - Routes, Vehicles, Drivers, Allocation, Fees
9. **Hostel** - Rooms, Attendance, Mess, Fees
10. **Communication** - Announcements, Notifications, Messaging, Notice Board
11. **Inventory** - Assets, Supplies, Maintenance, Vendors
12. **HR & Payroll** - Salary, Processing, Tax, Payslips, Integration
13. **Reports** - Student, Attendance, Financial, Staff, Custom, Export
14. **Settings** - Profile, Academic Year, Roles, Preferences, Backup
